# ContactsApp
Javafx beginner contact App

This is a basic  JavaFx contacts app.

## RUN

Simply compile the project and run to bring up the GUI. 

## Functionality

- The add button clears all the fields to allow for new input of new contact.

- To add a picture simply click the blank area and choose picture file from window.

- The save button adds a contact and its info to the list of contacts.

- You can also select a contact to delete and clicking the delete button.

- Clicking any contact will bring up the full information.

- Export button allows to export contact information into an xml file of your choice.

- Import button allows import of xml file into GUI.
